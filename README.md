# SimpleGeoApp
Simple Geolocation app for react native
## API Key
You will need to add a valid API key to the utils/API.tsx line 50 after key.

# Running instructions
## Step One: Copy
Clone repo
## Step Two: Install dependencies
```yarn install```
## Step Three: Instal Pods
```npx pod-install```
or
```cd && pod install```
## Step Four: Run
```yarn ios```
